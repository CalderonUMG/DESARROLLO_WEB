Esta página web fue creada utilizando HTML basico

HTML es el lenguaje principal para crear páginas web. Es la estructura sobre la cual se construye cualquier sitio, desde los más simples hasta los más complejos. Aprender HTML básico te permite comprender cómo se organizan los contenidos en una página: textos, imágenes, enlaces, listas, tablas, formularios, etc.
